Inversor String vs Microinversores

Entenda visualmente a diferença entre as duas principais tecnologias de geração solar.

Sistema com Inversor String

Os painéis são conectados em série, enviando toda a energia para um único inversor central. É a solução ideal para telhados amplos, sem sombras e que buscam o melhor custo-benefício em instalações simplificadas.

Sistema com Microinversores

Cada painel possui seu próprio inversor individual. Isso permite que cada placa opere de forma independente, garantindo que sombras ou sujeira em um painel não afetem a produção dos outros. Máxima eficiência para telhados complexos. Segurança máxima aprovada pelas novas normativas dos Bombeiros.
